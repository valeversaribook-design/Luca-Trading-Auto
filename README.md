# Luca Trading Auto

Versione Next.js pronta per GitHub + Vercel.

## Funzioni
- CSV TradingView/OANDA
- Grafico con anteprima candela
- Scelta manuale apertura/chiusura
- Generazione automatica di screen multipli
- Vincoli: numero screen, positive, negative, profitto min/max, lotto min/max, range orario
- Download ZIP con PNG + CSV operazioni generate

## Avvio locale

```bash
npm install
npm run dev
```

Poi apri:

```text
http://localhost:3000
```
